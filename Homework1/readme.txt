
Чтобы запустить программу нужно вписать в консоль/терминал следующую команду "node index.js X Y Z" - Где X,Y,Z - коэффиценты квадратного уравнения.

Если не ввести коэффиценты, то консоль выдаст следующее уведомление: 
"Введите коэффиценты квадратного уравнения через аргументы node js в следующем порядке: a, b, c - через пробел
Вы не ввели аргументы"

Если хоть один из аргументов будет не числом, а также если будет введено не 3 аргумента, то выдаст следующее сообщение:
> node index.js 1 2 q
"Вы ввели не целочисленные значения в аргументы или больше трёх аргументов"

Если дискриминант окажется больше нуля, то будет выдано следующее сообщение:
> node index.js 1 8 2
"Cуществует 2 корня уравнения"

Если дискриминант окажется равен нулю, то будет выдано следующее сообщение:
> node index.js 1 2 1
"Cуществует 1 корень уравнения"

Если дискриминант окажется меньше нуля, то будет выдано следующее сообщение:
> node index.js 2 2 2
"Корней уравнения не существует или их бесконечное множество"
